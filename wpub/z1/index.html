Podstránky

Domov - uvítacia stránka so základnými údajmi

O mne - stránka obsahujúca podrobnejšie informácie, záujmi a odkaz na recenzie

Recenzie kníh - stránka obsahuje zoznam mojich knižných recezií

Profesijné info - obsahuje moje počítačové zdatnosti a výber najzaujímavejších riešených projektov

CV - obsahuje CV v europass formáte a odkaz na jeho stiahnutie

Blog obsahuje príspevky blogu

Webové publikovanie - obsahuje odkazy na zadania z predmetu webové publikovanie

Layouty

default - základný layout použitý na všetkých podstránkach - ostatné layouty ho rozširujú.

post - layout na vykreslenie jednotlivých postov blogu

review - layout na vykreslenie jednotlivých recenzií na knihy

Kolekcie

_posts- obsahuje jednotlivé posty blogu

_work_experiences- obsahuje jednotlivé pracovné skúsenosti

Tagy

% include header- vkladá header

% include footer- vkladá footer

% for posts- vykreslí stráknu pre informácie z každého postu

%link -vytváral linky na stránky pri postoch a reviews

% youtube -vkladá youtube videá

Premenné

content- obsah

title- názov stránky

{{site.XXX}}- referencia stránky

author-názov autora knihy

rating- hodnotenie knihy

series= nazov kniznej serie do ktorej patrí recenzovana kniha

Pluginy

youtube- umoznuje prehrávať yooutube videá pomocou skráteného zápisu

Štýly
